C++ program to implement a class named Time

In this blog post, I will be explaining how to write a C++ program to implement a class named Time. This class should have a constructor to initialize the time, hours, minutes and seconds. The class should also have another function named ToSecond() to convert and display the time into seconds.

The following code snippet will show how to write a C++ program to accomplish this task. 

#include <iostream>
using namespace std;

class Time
{
    private:
        int hours;
        int minutes;
        int seconds;
    public:
        // Constructor to initialize the time
        Time(int h, int m, int s)
        {
            this->hours = h;
            this->minutes = m;
            this->seconds = s;
        }

        // Function to convert and display the time into seconds
        void ToSecond()
        {
            int timeInSeconds = (hours*3600) + (minutes*60) + seconds;
            cout << "Time in seconds is: " << timeInSeconds << endl;
        }
};

int main()
{
    Time t(2, 30, 15);
    t.ToSecond();
    return 0;
}

In this program, a class named Time is created which has three private variables to store the hours, minutes and seconds of the time. The constructor is used to initialize the time. The function ToSecond() is used to convert the time into seconds and display it.

In the main() method, an object of the class Time is created and the values for hours, minutes and seconds are passed. The function ToSecond() is called to convert and display the time into seconds.
