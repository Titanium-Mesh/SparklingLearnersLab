Creating your first HTML web page is easier than you think!

First, open up a text editor and create an HTML file. You can use Notepad for Windows or TextEdit for Mac. Next, type the following code at the top of your file: 

            
            <!DOCTYPE html>
            <html>
This code tells the web browser that the document is written in HTML. Then, after the tag, add the following code: 
            
            <head>
            <title> My First Webpage </title>
            </head>
This code creates the head section of the HTML document. Inside the head section, you can add a title for your webpage. Now, after the head section, add the following code: 
 
            <body>
            <h1>Welcome to My First Webpage!</h1>
            <p>This is my first HTML webpage.</p>
            </body>
This code creates the body section of the HTML document. Inside the body section, you can add any content that you want to display on the web page. In this example, we've added a heading and a paragraph. Finally, after the body section, add the following code to close the HTML document: 
            </html>
That's it! You've successfully created your first HTML web page. You can now save this file, open it in a web browser, and view your web page. Congratulations!
