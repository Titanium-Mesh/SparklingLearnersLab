C++ Program to Calculate the Area of a Circle

In this tutorial, we will learn how to write a C++ program to calculate the area of a circle using a class named Circle which has two functions with the names: GetRadius and CalArea.

Explanation

Firstly, we need to include the necessary libraries:

#include<iostream>
#include<conio.h>

Next, we create a class named 'Circle' which contains the two functions GetRadius() and CalArea():

class Circle
{
  private:
    float radius;
  public:
    //Function to get the value of radius from user 
    void GetRadius()
    {
        cout<<"Enter the radius of the circle: ";
        cin>>radius;
       
    }
    //Function to calculate the area of circle
    void CalArea()
    {
        float area = 3.14*radius*radius;
        //Display the area of the circle
        cout<<"The area of the circle is: "<<area<<endl;
        
    }
};

Finally, we define the main function where we create an object of the class 'Circle' and call the two functions GetRadius() and CalArea():

void main()
{
    Circle c;
    c.GetRadius();
    c.CalArea();
    getch();
}

This program will ask the user to enter the radius of the circle and then calculate the area of the circle and display the result.
